# Its My QR Code

Privacy-first, zero-backend QR code generator built with Next.js 14 and `qr-code-styling`.

## Features
- **100% Client-Side**: No data ever leaves your device. All QR generation happens in the browser.
- **PWA Ready**: Installable on mobile and handles offline usage.
- **Customization**:
  - Multiple input types (URL, Text, WiFi, etc.)
  - Real-time preview
  - Color gradients and solid colors
  - Custom dot patterns, eye frames, and eye balls
  - Logo embedding
- **High Resolution Export**: Download as PNG, SVG, or PDF.
- **2027 Compliant**: Built with "Equal Prominence" cookie banners and Easy-Exit subscription logic.

## Tech Stack
- Next.js 14+ (App Router)
- Tailwind CSS & Shadcn UI
- qr-code-styling
- Lucide React

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
